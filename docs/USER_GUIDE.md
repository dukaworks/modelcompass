### 第一步：访问平台

或本地开发环境：**http://localhost:3000**

### 第二步：浏览模型
无需登录，直接看：
1. 进入**模型广场** → 看到393+模型卡片
2. 点击**筛选** → 勾选"免费模型"

### 第三步：智能匹配
1. 点击顶部 **"AI匹配"** 菜单
2. 输入你的使用场景：  
   *"我需要处理大量中文法律文档，要求准确率高"*
3. 获得推荐 → 系统推荐3个最适合的模型（如：Qwen2.5-72B）
4. 点击**查看详情** → 了解为什么推荐这个模型

### 第四步：申请接入
1. 点击右上角 **"登录"**（GitHub一键登录）
2. 进入 **"API Keys"** 页面
3. 点击**创建Key** → 获得调用密钥
4. 复制代码示例 → 接入你的项目

## 🎯 核心功能详解

### 1. 模型广场 —— 你的AI模型超市

**功能**：浏览、筛选、对比所有可用模型

**使用技巧**：
- 🔍 **搜索**：输入"GPT"、"DeepSeek"快速找到
- 🏷️ **筛选**：按能力（对话/代码/多模态）、价格、上下文长度

### 2. AI智能匹配 —— 为你推荐最合适的模型

**适合场景**：
- 不知道选哪个模型
- 预算有限，想找性价比高的
- 特定场景（中文/代码/长文档）

**使用方法**：
1. 点击 **"AI匹配"** 进入对话界面
2. 描述你的需求（越详细越好）：
   - ✅ 好例子：*"我是电商客服，需要处理用户退换货咨询，要求中文理解好，响应快，预算有限"*
   - ❌ 差例子：*"推荐个模型"*
3. 查看推荐结果：
   - 主要推荐：核心场景匹配
   - 备选方案：性价比之选
   - 高端选项：预算充足时的选择
4. 点击**查看详情**或**直接体验**

**为什么这样推荐？**  
系统根据你的描述分析：
- 中文能力权重高 → 推荐Qwen/DeepSeek
- 预算敏感 → 排除GPT-4
- 客服场景 → 推荐对话优化模型

### 3. 统一API接入 —— 一行代码调用全球模型

**传统方式的问题**：
```python
# 要对接OpenAI
import openai
openai.api_key = "sk-xxx"

# 要对接DeepSeek
import requests
# 完全不同的接口...

# 要对接Qwen
# 又要改代码...
```

**ModelCompass方式**：
```python
# 一个接口，切换模型只需改名字
import requests

response = requests.post(
    "https://api.modelcompass.ai/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-4o",  # 改这里切换模型
        "messages": [{"role": "user", "content": "Hello!"}]
    }
)
```

**切换模型只需改名字**：
- `"model": "gpt-4o"` → OpenAI GPT-4o
- `"model": "deepseek-chat"` → DeepSeek-V3
- `"model": "qwen-2.5-72b"` → 阿里通义千问

**自动故障转移**：  
如果某个模型提供商故障，自动切换到备用提供商，你的应用不中断。
