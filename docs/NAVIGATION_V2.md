### 1. 申请 API (统一 API 服务)
**定位**: 对外提供标准化 API 接入
**页面内容**:
- API 能力介绍 (统一接口调用多模型)
- 价格体系 (按量计费 / 套餐)
- 快速接入指南
- 在线测试工具
- 【重点】无需登录即可查看文档和定价

**URL**: `/api-service` 或 `/pricing`

---

### 2. 模型市场 (模型广场)
**定位**: 类似 "轨迹流动" 的模型广场
**理解**: 应该是瀑布流/卡片式展示所有模型，支持筛选

**页面内容**:
- 🎴 模型卡片瀑布流 (图片 + 名称 + 能力标签)
- 🔍 多维度筛选 (提供商/能力/价格/上下文)
- 📊 实时排行榜 (周榜/月榜)
- ⭐ 社区评分显示
- 💬 模型评价数

**URL**: `/market` 或 `/models`

**区别原设计**:
- 原「模型库」→ 现在「模型市场」更市场化
- 增加社区互动元素 (评分、评论)

---

### 3. AI 匹配对话 (核心功能)
**定位**: OpenWebUI 对话界面 + 智能推荐代理
**目标**:
- ✅ 提供免费模型进行对话体验
- ✅ 通过对话理解用户需求
- ✅ 后台智能匹配推荐模型
- ✅ 采集用户知识/反馈

**页面结构**:
```
/recommend (或 /chat)
├── 左侧: 对话历史
├── 中间: 
│   ├── 🤖 AI 助手: "你好！我是 ModelCompass 助手，告诉我你的使用场景..."
│   ├── 💬 免费对话区 (接入轻量级模型 Qwen-1.8B 等)
│   └── 📊 右侧推荐栏 (根据对话实时推荐模型)
└── 底部: 输入框 + 功能按钮

对话流程示例:
1. 👤: "我需要处理大量中文法律文档"
2. 🤖: "理解您的需求了，推荐以下模型：" [卡片展示]
   - DeepSeek-V3 (法律推理强)
   - Qwen-2.5-72B (中文优化)
   - GPT-4o (通用最强)
3. 👤: "哪个性价比最高？"
4. 🤖: [展示价格对比表] + 推荐结论
5. 👤: [点击"测试这个模型"] → 跳转实际调用
```

**URL**: `/chat` 或 `/recommend` 或 `/ai-assistant`
