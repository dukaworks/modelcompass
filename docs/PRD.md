## 1. 产品概述

### 1.1 产品名称
**ModelCompass (魔盘)** - AI模型比价与推荐平台

### 1.2 一句话描述
帮助开发者找到性价比最高的AI大模型，省钱又省心。

### 1.3 目标用户
- **主要**: 中文开发者（全栈/后端/AI应用开发）
- **次要**: 技术团队负责人、AI产品经理
- **场景**: 选型时对比价格、寻找替代方案、控制API成本

---

## 2. 核心痛点

| 痛点 | 现状 | 解决方案 |
|------|------|----------|
| 模型太多不知道选哪个 | OpenAI贵，国产多，对比困难 | AI智能匹配，输入场景自动推荐 |
| 价格不透明 | 各家计费方式不同 | 统一比价，¥/M tokens标准化 |
| 免费模型难找 | 隐藏在各种文档里 | 一键筛选免费模型 |
| 中文场景不适配 | GPT-4中文不如国产模型 | 中文场景特别推荐 |

---

## 3. 功能架构

### 3.1 核心功能 (P0 - 必须有)

#### 3.1.1 模型广场
- 展示所有模型卡片（硅基流动风格）
- 筛选：免费/付费、能力类型、上下文长度
- 排序：价格/热度/评分
- 搜索：模型名、提供商

#### 3.1.2 AI智能匹配
- 对话式推荐：描述场景 → 推荐3个最适合模型
- 推荐理由：价格优势/中文能力/代码能力
- 直接跳转：体验/API接入

#### 3.1.3 统一API接入
- 一个接口调用全球模型
- 兼容OpenAI格式
- 自动故障转移

#### 3.1.4 用户系统
- 注册/登录（GitHub OAuth）
- API Keys管理
- 余额/账单

### 3.2 增强功能 (P1 - 体验好)

- 模型收藏/对比
- 调用统计Dashboard
- 社区评测/评论
- 价格变动提醒

### 3.3 未来功能 (P2 - 锦上添花)

- 企业级：团队协作、预算管控
- 开发者：Prompt模板库、微调服务
- 移动端App

## 4. 技术架构

### 4.1 前端
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS + 深色科技风
- **状态**: React hooks + localStorage

### 4.2 后端
- **运行时**: Node.js + Express
- **数据库**: PostgreSQL + Prisma ORM
- **爬虫**: 定时抓取OpenRouter/HuggingFace

### 4.3 部署
- **开发**: Docker Compose
- **生产**: Railway + Vercel

## 5. 用户旅程

### 5.1 首次访问（浏览模式）
```
1. 进入首页 → 看到搜索框 + 统计数字（60+提供商, 393+模型）
2. 浏览模型广场 → 筛选免费模型 / 看价格对比
3. 点击"AI匹配" → 描述场景（如：中文客服）
4. 获得推荐 → 看到DeepSeek/Qwen等中文友好模型
5. 感兴趣 → 注册登录
```

### 5.2 注册用户（使用模式）
```
1. 登录 → 进入AI匹配对话 / 直接浏览模型
2. 找到合适模型 → 申请API Key
3. 查看文档 → 复制代码示例
4. 接入项目 → 开始调用
5. 查看账单 → 监控用量和费用
```

### 5.3 付费用户（高级模式）
```
1. 充值余额 → 微信/支付宝
2. 获得更高配额
3. 使用高级功能（批量推理、微调）
4. 企业功能（团队管理）
```

## 7. 商业模式

### 7.1 收入来源
- **API调用差价**（主要）：统一接口，加价10-20%
- **会员订阅**：高级功能（更多调用量、优先支持）
- **企业版**：私有化部署、定制模型

### 7.2 免费策略
- 免费模型：调用免费，我们承担成本（获客）
- 付费模型：正常计费，我们抽成
- 免费额度：新用户送¥50试用
